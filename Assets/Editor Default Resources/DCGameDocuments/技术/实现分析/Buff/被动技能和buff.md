## 被动技能的分类
- 附加伤害
- 修改数值
- 每几次攻击触发额外效果
  - 治疗
  - 数值

## 比较被动技能是否可以用buff实现
### 用技能对象实现被动技能
- 方式1
```
创建一个跟随施法者的技能对象
如果上一次的受击者离开区域，移除buff
收集本次受击者
如果受击者没有buff，加上buff
```

### 用buff实现被动技能
```
被动技能生效的时候创建一个buff放在主角身上
如果上一次的受击者离开区域，移除buff
收集本次受击者
如果受击者没有buff，加上buff
需要配置buff的包围盒
```
